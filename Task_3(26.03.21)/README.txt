Задание_2:
	Сценарий: Вывод списка сохраненных форматов файлов
	Дано: Программа выводит меню со списком и соответствующих ему разделов меню
	Когда: Пользователь выбирает первый пункт меню, нажимая клавишу 1.
	То: Программа выводит список хранимых форматов файлов, состоящих из <короткого> и <полного> наименования.

	Примеры:
	|Клавиша|	Раздел                         |
	|1      |Просмотр всех записей		       |
	|2      |Добавление записи           	       |
	|3      |Удаление записи(по короткому описанию)|
	|4      |Поиск записи(по короткому описанию)   |
	|5      |Выход			               |


	Cценарий: Ввод несуществующего пункта.
	Дано: Программа просит нажать клавишу соответствующую пункту меню.
	Когда: Пользователь нажимает клавишу, которая не привязана ни к одному пункту меню.
	То: Программа отчистит экран, выведет меню и снова попросит выбрать пункт меню.


	Сценарий: Добавление нового формата в список.
	Дано: Программа просит ввести короткое наименование формата, потом просит ввести полное наименование.
	Когда: Пользователь ввел не пустые строки.
	То: Данный формат будет добавлен в конец списка.


	Сценарий: Добавление существующего формата в список.
	Дано: Программа просит ввести короткое наименование формата, потом просит ввести полное наименование.
	Когда: В списке будет найдена запись с таким ключом.
	То: Программа выведет сообщение "Ошибка!".


	Cценарий: Добавление формата с пустым кратким или полным наименованием.
	Дано: Программа просит ввести короткое наименование формата, потом просит ввести полное наименование.
	Когда: Пользователь ввел все параметры.
	То: Программа выведет сообщение "Ошибка! Поля не могут быть пустыми!"


	Сценарий: Удаление формата из списка по краткому наименованию.
	Дано: Программа просит ввести короткое наименование формата.
	Когда: Пользователь ввел краткое наименование.
	То: Из списка удаляется запись с таким кратким наименованием.

	
	Сценарий: Удаление формата из списка несуществующего формата.
	Дано: Программа просит ввести короткое наименование формата.
	Когда: Пользователь ввел краткое наименование несуществующего формата.
	То: Программа выводит сообщение "Ошибка! Данный объект не найден!".


	Сценарий: Поиск элемента по краткому наименованию.
	Дано: Программа просит ввести короткое наименование формата.
	Когда: Пользователь ввел краткое наименование формата.
	То: Программа выводит сообщение краткое и полное наименование.


	Сценарий: Поиск несуществующего элемента по краткому наименованию.
	Дано: Программа просит ввести короткое наименование формата.
	Когда: Пользователь ввел краткое наименование формата.
	То: Программа выводит сообщение "Ошибка! Данный объект не найден!".
